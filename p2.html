STUCTO (To-Do-List Python App)

"STUCTO" The app is designed to keep users committed to their daily goals. This project aims to provide a tool to help users manage their daily routines effectively.

I'm using the CustomTkinter library in Python for the GUI, and this is the initial look of the application.

This application is designed to allow users to add, edit, and view a list of items they have saved.

This application has great potential. I envision adding features like a calendar and automatic reminders to make it even more useful for users.

For developers, I'm using MySQL as the database management system for this application, which can be manipulated automatically or manually depending on the developer's needs.
