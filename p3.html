Handwritten Digit Recognition

This is my third project, a handwritten recognition system developed using Python programming language.

Here, the development utilizes the Keras library, which already includes several datasets, with MNIST being one of them.

Although it still has many shortcomings, I hope to develop this into a more versatile project to assist users.

It can be observed that there are still errors in digit interpretation in this project. My goal moving forward is to develop this project to enhance user efficiency.
