EAZYDINE (Food Ordering App)

In this group project, my role is to design the application and manage its usability. This project utilizes object-oriented programming in Visual Basic 6 to develop a food machine application that facilitates online food ordering.

Eazydine is a technology-based food ordering application that offers convenience in placing food orders. It features three customizable menus.

After placing an order, users can proceed to the payment page as described above. The application will automatically calculate the total amount due.

Although this application utilizes older programming languages, it has proven to be user-friendly and practical.
